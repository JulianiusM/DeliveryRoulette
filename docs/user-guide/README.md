## Recommended First-Time Path

1. Create an account and sign in.
2. Save a location in **Settings**.
3. Open **Suggest** and try a draw immediately.
4. If the app finds matching restaurants already in the database, use them.
5. If the app reports missing restaurants for your location, open **Location Imports** to refresh or discover them.
6. Use **Restaurants** to review the shared catalog.
